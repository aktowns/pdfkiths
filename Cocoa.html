Treat NSBool as a CInt (that should be either 1 | 0). It seems most of the NS frameworks use this, instead of C99 booleans.

startRunloop :: IO ()

This is required to make obj-c calls work enables the default runloop, this call locks the thread its run in, so its advised to forkIO it. This is the equivalent to [[NSRunloop mainRunLoop] run];
